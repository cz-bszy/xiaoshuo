# 《西幻种田：穿越者的领地崛起》总纲

## 核心信息

- **类型**：西方幻想、种田/领地建设、系统流、军政
- **目标字数**：450万字
- **卷数**：5卷，每卷约90万字（180章×5000字）
- **总章节**：约900章

---

## 核心主线

### 终极目标
主角从一无所有的穿越者，成长为拥有强大领地、精锐军队、深厚底蕴的一方霸主，在这个魔兽与人类共存的世界建立自己的传世基业。

### 发展主线
```
阶段1：生存立足 → 阶段2：领地建设 → 阶段3：势力扩张 → 阶段4：王国争霸 → 阶段5：霸业终成
```

### 个人成长线
```
无境界 → 初潮者 → 涌潮者 → 浪涛者 → 潮主 → 海眼 → (接近渊主)
```

## 世界观核心

### 地理格局

**大陆名称**：诺恩大陆

主要势力：
- **洛兰王国**（主角所在）：老牌王国，内部派系复杂
- **铁蔷薇帝国**：军事强国，北方威胁
- **自由城邦联盟**：商业发达，东部
- **荒野领地**：边境，魔兽频繁，主角起步地
- **深渊森林**：高级魔兽栖息地

### 魔力体系——生命潮汐

**核心概念**：
- 世界存在基础能量"源力"
- 生物通过吸收源力强化，称为"潮汐"
- 潮汐如潮水涨落，源力在体内循环积累

**六大境界**：
1. **初潮者**：感知源力，体质增强（普通士兵上限）
2. **涌潮者**：稳定循环，力量倍增（精锐级别）
3. **浪涛者**：外放源力，战斗核心（小领主级）
4. **潮主**：掌控一域，战略级别（大贵族/伯爵级）
5. **海眼**：源力凝实，超越凡人（公爵/军团级）
6. **渊主**：传说存在，一国之柱（王国级）

### 魔兽体系

魔兽同样通过潮汐进化，分为六级：
- 一二级：村庄威胁，猎人可处理
- 三四级：军队级威胁，需要正规军
- 五六级：战略级，需要高阶强者

### 社会制度

中世纪封建制：
- 国王→公爵→侯爵→伯爵→子爵→男爵→骑士领
- 贵族领地制，领主拥有领地治权
- 农奴和自由民共存
- 教会存在但影响有限

## 主角设定

### 基础信息
- **穿越前**：林远，25岁，城市规划师
- **穿越后**：艾伦·诺斯，边境小贵族庶子
- **性格**：务实、规划者、果断、适应力强

### 金手指——星辰系统

**功能**：抽奖池系统
**获取方式**：完成任务获得"星尘"
**奖励等级**：白→绿→蓝→紫→金
**核心限制**：
- 每日抽奖次数有限
- 高级奖励概率极低
- 奖励需要自己开发利用
- 系统不提供指导

### 成长路线
| 卷数 | 境界 | 势力规模 |
|------|------|---------|
| 第一卷 | 无→涌潮者 | 获得一块领地 |
| 第二卷 | 涌潮者→浪涛者 | 稳固领地 |
| 第三卷 | 浪涛者→潮主 | 地区势力 |
| 第四卷 | 潮主→海眼 | 王国级势力 |
| 第五卷 | 海眼→ | 霸业完成 |

主角成功建立强大势力，在大陆格局中占据重要地位。无论是领地建设、军事力量还是个人实力，都达到了令人敬畏的高度。留下可以传承的体系和基业，故事圆满收尾。
